# Assets

Put your app images here. The following files are referenced in `app.json`:

- `icon.png` — 1024x1024 app icon (square, no transparency)
- `splash-icon.png` — splash screen image (centered, transparent background OK)
- `adaptive-icon.png` — Android adaptive icon foreground (1024x1024)
- `favicon.png` — web favicon (48x48 or larger)

If any of these are missing, Expo will log warnings but the app will still run in development.

You can delete this README once real assets are in place.
